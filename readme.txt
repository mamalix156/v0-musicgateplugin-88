Premium subscription system with content restriction overlay and Zarinpal payment integration for WordPress.

== Description ==

Music Gate is a comprehensive WordPress plugin that provides a premium subscription system with content restriction capabilities. It features a modern, music-themed design and integrates seamlessly with Zarinpal payment gateway for Iranian users.

**Key Features:**

* **Content Restriction**: Automatically restricts access to posts and pages with customizable overlay
* **Zarinpal Integration**: Secure payment processing with Iran's leading payment gateway
* **User Management**: Simple registration with minimal required fields (name and phone only)
* **Subscription Plans**: Flexible 1-month, 3-month, and 1-year subscription options
* **Modern UI**: Beautiful, responsive design with music-themed aesthetics
* **Admin Dashboard**: Complete management system for subscriptions, orders, and settings
* **Shortcode Support**: Easy integration with existing content using shortcodes

**Shortcodes:**

* `[musicgate_register]` - Display registration form
* `[musicgate_buy_button plan="month1"]` - Show subscription purchase button
* `[musicgate_user_dropdown]` - User profile dropdown for header

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/music-gate/` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Music Gate settings to configure your subscription plans and payment settings
4. Add shortcodes to your pages where needed

== Frequently Asked Questions ==

= How do I set up Zarinpal payments? =

Go to Music Gate > Settings and enter your Zarinpal Merchant ID. The default ID is already configured for testing.

= Can I customize the restriction percentage? =

Yes, you can set any percentage from 0-100% in the plugin settings to control how much content is hidden.

= What user information is required for registration? =

Only first name, last name, and phone number are required. No email verification needed.
